# 16 Personalities Evaluation App

A self-contained browser app for an educational MBTI-style 16 personality evaluation.

## Open the app

Open `16-personalities.html` in any modern browser, or preview it directly from the workspace.

## Features

- 48-question Likert assessment
- Scoring for four dimensions: E/I, S/N, T/F, J/P
- 16 type result profiles with strengths, growth tips, and energizing environments
- Preference-balance bars and balanced-dimension notes
- Progress saved locally in the browser
- Theme toggle, keyboard shortcuts, copy summary, JSON download, and print/PDF support

## Disclaimer

This is an educational reflection tool inspired by Jungian/MBTI-style type theory. It is not the official MBTI instrument and should not be used for diagnosis, selection, or hiring decisions.
